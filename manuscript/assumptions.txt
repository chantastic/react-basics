# Assumptions

* You **know** elementary HTML.
* You **know** elementary JavaScript.
* You **know** what the DOM is.
* You **don't** know anything about React.js.
* You **have** an interest in learning React and not gaining jargon for nerd cocktail parties.

React can be used anywhere.

My goal is to get into React as quickly as possible. For that reason, I'm start with an HTML file and a few script tags.

Once you've learned the basics of React, find out insert into your desired platform. Chances are someone has already figured out how.
