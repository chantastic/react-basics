# getInitialState

## NOTES

* Components created with createClass can keep local state.
* initial state is provided through getInitialState. it's a function that returns a state object.
* we can put anything we want in this object
* we can use the value in our function...
* ... and interpolate it in JSX for DOM output

## RESOURCES

## CODE

``` js
var ClickCounter = React.creatClass({
  getInitialState: function () {
    return { clicks: 0 }
  },

  handleClick: function () {
     alert(0 + 1)
  },

  render: function () {
    return <div onClick={this.handleClick}>
      clicks: 0
    </div>
  }
})

ReactDOM.render(
  <ClickCounter/>,
  document.getElementById("root")
)
```

``` js
var ClickCounter = React.creatClass({
  getInitialState: function () {
    return { clicks: 0 }
  },

  handleClick: function () {
     alert(this.state.clicks + 1)
  },

  render: function () {
    return <div onClick={this.handleClick}>
      clicks: {this.state.clicks}
    </div>
  }
})

ReactDOM.render(
  <ClickCounter />,
  document.getElementById("root")
)
```
